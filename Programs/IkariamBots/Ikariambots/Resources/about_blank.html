What is Ikariam+?

Basic information
Ikariam+ is a program to your computer that allow you to connect to your ikariam's account, on any server, on any country.
You can open multiple windows on your multiple accounts (Just run Launcher.exe again)
So what? It's not a big deal, I can open multiple windows in my browser
The short answer is yes. You can.
But, in Ikariam+ you can use in one of the many build-in tools that can improve your game's experience!
Tools like Ikariam+ Build List that give you infinite times of buildings to build.
Or Ikariam+ Piracy Bot that can do to you all the pirates missions, in minimum time, auto convert raid points to crew points and auto solving captcha! Indeed, it can solve to you the annoying captcha automatically
If you don't like to browse from your IP, you can use proxy (in the Launcher.exe) in no time, just write the proxy address and port and the program will use it over your address! For example to proxy: 1.2.3.4:8080
Tor is included, you can run it from launcher (make sure that ports 9158 and 9058 are free)
Also, we protect your data, all the connection are on the SSL server (https) by default, and there is an option to use TOR as an sockets proxy bridge, as default write in the proxy address: socks://127.0.0.1:9150 while tor is started!
Ikariam+ comes with simple launcher, so you don't have to do complex things to start
Also, the program will auto update every time there is a patch/update for the program. So if there is a bug update/problem solved/new feature, you'll get it.
Here is a full list of all the tools in Ikariam+:

Ikariam+ Build List

Ikariam+ Build List is a tool that you can use for building several structures in queue, for infinite length, unlike the one you buy in ambrosia.
How to use:
1) On any structure click on "Add to queue".
2) Repeat on level 1 until you've done you build list.
3) Go to Scripts -> Build List -> Start Queue.

Ikariam+ Piracy Bot

Ikariam+ Piracy Bot is a tool that do to you the piracy missions. Meaning, you don't need to click every 2.5 minutes.
This tool works on all the missions, and have many more options, like auto convert all the capture points into crew points,
or, auto solving captcha (with a special service, it doesn't cost extra money!), so you can leave the program alone and watch your points rise and rise.
(And you can run it on multiple accounts!)
There is a new option that you can run the piracy script in minimal requirements, so you can run even more accounts at the same time!
Just pick the option "Massive piracy use" in the launcher - If you use proxy, it's must be socks/web proxy (Like Tor)
Pay attention: if you'll do too much points per day, you'll get ban! Try to do 20,000 points max per day.
How to use:
1) Go to a "Massive piracy use" in the launcher.
2) Configure all the users.
3) Click "Login".
Captcha solving (9KW) 1) Register on 9kw.eu.
2) Create API access points at 9kw.eu/userapi.html.
3) Solve captcha for others at 9kw.eu/usercaptcha.html.
4) Run the program via Launcher.exe and set 9kw api key and 9kw api source from the data you set in step 2.

De-Captcher supported too!

Ikariam+ Spy

Ikariam+ Spy is a tool that sends you spies to select city and find for you the best spies/decoys ratio for your mission. Simple as that.
How to use:
1) Go to Scripts -> Spy -> Start Script.
2) Write the cities IDs (City ID can be seen in island view where the number near the city name is the ID) you want the spies to be sent splited by comma.
   For example: "16352,12345,54321" (Without spaces!)

Ikariam+ Military Calculator

Ikariam+ Military Calculator is a tool that can give you the exact generals/wood/wine/etc. that you'll get after creating the army.
How to use:
1) Go to Calculator -> Military calculator.

Ikariam+ Fast Locations

Ikariam+ Fast Locations is a tool that you can use for jumping to selected locations in no time (yes, there is an option, but is limited by maximum locations).
And it's with unlimited locations for your choice!
How to use:
1) Edit config.xml from Ikariam+'s main folder.
2) Add Like the example under Bot/FastLocations.

Ikariam+ Islands By Time

Ikariam+ Islands By Time is a tool that can give you all the posible islands by given time and start points.
The spy in ikariam can spy after troops movement, but it only shows start and arrive time. With Ikariam+ Islands By Time you can find the island that the troops will be at the end time!
How to use:
1) Go To Calculator -> Islands by time.

Ikariam+ Search Users/Allies/Cities

Ikariam+ Search Users/Allies/Cities is a tool that can give you full list in a few minutes on all the locations where a user/city/ally is located. Including occupation cities and occupation ports.
This tool also exports all the data to CSV file (Excel table) with information like: Wonder and level, what is the buildings level in this town (if you have spies/right to see the city). How to use:
1) Go to Scripts -> Search -> Search by ally tag/player name/city name.
2) Wait until complete.
3) Go to Ikariam+'s main folder and open the file Report*DAY**MONTH**YEAR*-*HOUR**MINUTE*.
* If you have a spy/right to see a city, all the buildings in this city will add to the report.

Ikariam+ Capture Reports

Ikariam+ Capture Reports is a tool that saves in your computer your battle reports/spy reports in a click of a button.
How to use:
1) Open report (spy/military report).
2) Go to Scripts -> Capture report -> Capture ... report.

Ikariam+ Attacks Alert

Ikariam+ Attacks Alert is a tool that alerts you (by annoying beep) when you/some in your ally(if your'e the general) is atacked.
When you attacked you can choose to auto active the Ikariam+ Scripted Attack option!
How to use:
1) Go to Scripts -> Check for attacks -> Check for attack on me/ally(embassy must be open).
2) When there is attack on you only Ikariam+ Scripted Attack will start automatically with the file "AttackedFight.ikr".

Ikariam+ Scripted Attack

Ikariam+ Scripted Attack is a tool that attack (pillage raid/piracy raid/deploy units/etc.) by scripted order.
For example, if you write:
Set Hoplite 90
Set Ram 15
Raid
Wait 15
PiracyRaid
Set Hoplite 0
Set Ram 0
Set SteamGiant 200
Raid

It'll do:
* Configure to 90 hoplites and 15 rams
* Raid with the configured army
* Wait 15 minutes
* Piracy raid
* Configure to 0 hoplites, 0 rams and 200 steam giants
* Raid with the configured army
So easy, so you can ride others without even be there!
How to use:
1) Go to Scripts -> Scripted attack -> New scripted attack (for new file) / Start script (open exist file)

How to write a script

Every line should contain only 1 command.
Do NOT take scripts from sources you dont trust!
Writing "#" or ";" in the beginning of the line will be a comment and be ignored.
For example:
; This is a comment
# This is a comment too

Full commands list (without quotes)

  1. "GoTo *Line*" - The script return to line *Line* (replace with positive integer) and start from there (every positive integer is an option)
    For example:
    GoTo 1
    # The script will start from the beginning
  2. "Wait *Minutes*" - Delay the program for *Minutes* minutes(replace with positive integer)
    For example:
    Wait 10
    # The script will wait 10 minutes
  3. "RefreshData" - Refresh window data (same refresh that happens once in a minute)
    For example:
    RefreshData
    # The page will refresh
  4. "ChangeCityID *CityID*" - Change selected city.
    By that option the system will know to what city do you mean when you want to pillage/raid/etc.
    For getting the city ID you can see it in the island view, near the city name there will be unique number, that number is the city ID.
    For example:
    ChangeCityID 86615
    # The selected city will set to city ID 86615
  5. "ChangeCurrentCityID *CityID*" - Change current city.
    By that option the system will know to from what city you mean when you want to pillage/raid/etc.
    For getting the city ID you can see it in the island view, near the city name there will be unique number, that number is the city ID.
    This city must be by you control, meaning you can see this city in the city selector.
    For example:
    ChangeCurrentCityID 86615
    # The current city will set to city ID 86615
  6. "UpgradeBuilding *PositionID*" - Upgrade a building by his position.
    The position will be shown in the city view next to the building name.
    For example:
    UpgradeBuilding 17
    # Upgrade pirate fortress
  7. "StartPiracy" - Start the piracy script (need to be on city view and with pirate fortress in this city).
    For example:
    StartPiracy
    # Start piracy script
  8. "StopPiracy" - Stop the piracy script.
    For example:
    StopPiracy
    # Stop piracy script
  9. "Raid" - Raid the selected city.
    For example:
    Raid
    # Raid the selected city
  10. "Occupy" - Occupy the selected city.
    For example:
    Occupy
    # Occupy the selected city
  11. "PiracyRaid" - Piracy raid the selected city.
    For example:
    PiracyRaid
    # Piracy raid the selected city
  12. "DeployUnits" - Deploy units into the selected city.
    For example:
    DeployUnits
    # Deploy units into the selected city
  13. "ChangeView *View*" - Change current view.
    *View* can be: "City", "Island", "World"
    For example:
    ChangeView Island
    # Switch to island view
  14. "JS *Command*" - Run JavaScript command (Only use it if you know what you are doing!).
    For example:
    JS alert('this script work');
    # Run JavaScript pop-up message
  15. "Set *Unit* *HowMuch*" - Set selected unit to battle/deploy/etc.
    *HowMuch* - (replace with positive integer)
    *Unit* - Hoplite/SteamGiant/Spearman/Slinger/Swordsman/Archer/SulphurCarabineer/Ram/Catapult/Mortar/Gyrocopter/BalloonBombardier/Cook/Doctor/TransporterShips
    For example:
    Set Hoplite 100
    Set Mortar 30
    Set TransporterShips 50
    # Set to 100 hoplites, 30 mortars and 50 transporter ships
  16. "ResetUnits" - Reset selected units to 0.
    For example:
    ResetUnits
    # Equal to "Set *Unit* 0" for all units

מה זה Ikariam+?

מידע בסיסי
Ikariam+ זו תוכנה למחשב שמאפשרת לך להתחבר לחשבון שלך בכל שרת ומדינה.
אתה יכול לפתוח כמה חשבונות במגביל (רק תריץ את Launcher.exe שוב)
מה הסיפור? זה לא ביג דיל, אני יכול לפתוח כמה כרטיסיות בדפדפן שלי
התשובה הקצרה היא כן, אתה יכול.
אבל, ב Ikariam+ אתה יכול להשתמש בכל אחת מהתכונות המובנות שמשפרות לך את חווית המשחק!
כלים כמו Ikariam+ רשימת בנייה שנותן לך רשימת בנייה אין סופית למבנים.
או Ikariam+ בוט פיראטים שעושה לך את כל המשימות, במינימום זמן, עם אפשרות להמיר נקודות לכידה לכוח צוות עם אפילו פתירת קאפצ'ה אוטומטית! אכן, זה יכול לפתור לך את הקאפצ'ות המעצבות ברגע!
אם אתה לא אוהב לגלוש מהIP שלך אתה יכול להשתמש בפרוקסי (מה-Launcher.exe) במהרה, רק תכתוב את כתובת הפרוקסי שלך והתוכנה תשתמש בכתובת זו! דוגמה לפרוקסי: 1.2.3.4:8080
Tor כלול, אתה יכול להריץ אותו מהדף פתיחה (אנא ודא שהפורטים 9158 ו 9058 פנוים)
בנוסף, אני מגנים על המידע שלך, כל החיבורים נעשים על שרת SSL מאובטח (https) כברירת מחדל, וישנה אפשרות להשתמש ב-Tor כגשר פרוקסי, כברירת מחדל לפרוקסי של Tor תכתוב את הכתובת: socks://127.0.0.1:9150 תוך כדי שTor רץ!
Ikariam+ מגיע עם פתיחה נוחה, ככה שאתה לא צריך לעשות הגדרות מסובכות כדי להתחיל
בנוסף, התוכנה תתעדכן אוטומטית בכל פעם שישנו עדכון/תיקון בעיה/תוסף חדש. ככה שאם יש תיקון/תוסף חדש/אפשרות נוספת, אתה תקבל אותו!.

Ikariam+ רשימת בנייה הוא כלי שנותן לך לבנות מספר מבנים ברשימה, לאורך אין סופי, בניגוד לאחד שאתה קונה באמברוסיה.

1) על כל מבנה ליד השם תלחץ על "Add to queue".
2) חזור על שלב 1 אחרי שסיימת את הרשימה.
3) לך ל Scripts -> Build List -> Start Queue.

Ikariam+ בוט פיראטים

Ikariam+ בוט פיראטים זה כלי שעושה לך את כל המשימות פיראטים. כלומר, אתה לא צריך להקליק כל 2.5 דקות.
הכלי הזה עובד על כל המשימות, ויש לו עוד הרבה שימושים, כמו להמיר את כל הנקודות לכידה לכוח צוות,
או, פתירת קאפצ'ה אוטומטית (עם שירות מיוחד, זה לא עולה עוד כסף - זה בחינם!), אז אתה יכול לעזוב את התוכנית פועלת ולראות את הנקודות רק עולות ועולות.
(בנוסף, אתה יכול להריץ את זה על כמה חשבונות במקביל!)
ישנה אפשרות להריץ את הבוט פיראטים במינימום דרישות מערכת, ככה שאתה תוכל להריץ אפילו יותר משתמשים כפולים!
רק תבחר באפשרות "פיראטיות" בדף הרצה - אם אתה משתמש בפרוקסי, זה חייב להיות socks פרוקסי (כמו Tor)
שים לב: אם תייצר יותר מידי נקודות ביום אחד, אתה תקבל באן! תנסה לא לעבור את רף ה-20,000 נקודות ביום.
פתירת קאפצ'ה (9KW) 1) הרשם ב-9kw.eu.
2) צור מפתח API ב- 9kw.eu/userapi.html.
3) פתור קאפצ'ות לאחרים ב-9kw.eu/usercaptcha.html.
4) הרץ את התוכנה דרך Launcher.exe והגדר את 9kw api key ו- 9kw api source מהנתונים שקיבלת בצעד 2.

De-Captcher נתמך גם!

Ikariam+ ריגול

Ikariam+ ריגול הוא כלי ששולח לך מרגלים למספר ערים נבחרות ובוחר את היחס הטוב ביותר של מרגלים/הסחות.
איך להשתמש:
1) לך לScripts -> Spy -> Start Script.
2) רשום את מספרי הזיהוי של הערים (מספר מזהה של עיר ניתן לראות במבט האי, כאשר המספר המזהה מופיע ליד שם העיר) שאתה רוצה שהמרגלים יישלחו מופרדות בפסיקים.
   לדוגמא: "16352,12345,54321" (בלי רווחיםs!)

איך להשתמש:
1) לך לCalculator -> Military calculator.

Ikariam+ מיקומים מהירים

אך אפשרות זו ללא הגבלה!
איך להשתמש:

Ikariam+ חיפוש שחקן/ברית/עיר

Ikariam+ חיפוש שחקן/ברית/עיר הוא כלי שמאפשר לך תוך כמה דקות לקבל רשימה של מיקומי עיר/שחקן/ברית כולל ערים/נמלים כבושים, רמת נס, איזה משאב יש באי וכ'ו.
הכלי בנוסף גם מיצא את כל המידע לקובץ CSV (טבלת Excel) עם מידע כמו: מה הפלא והרמה, מה המבנים באותה עיר (אם יש לך מרגלים/זכות לראות את העיר). איך להשתמש:
1) לך לScripts -> Search -> Search by ally tag/player name/city name.
2) חכה עד להשלמת החיפוש.
3) לך לתיקיית Ikariam+ ופתח את הקובץ Report*DAY**MONTH**YEAR*-*HOUR**MINUTE*.
* אם יש לך מרגל/זכות לראות את אותה עיר, רשימת המבנים בעיר תופיע בדו"ח

Ikariam+ צילום דוחות

Ikariam+ צילום דוחות זה כלי ששומר אצלך תמונות של דוחות מלחמה/ריגול בלחיצת כפתור
איך להשתמש:
1) פתח דוח (צבאי/ריגול).
2) לך לScripts -> Capture report -> Capture ... report.

Ikariam+ התרעת תקיפה

Ikariam+ התרעת תקיפה זה כלי שמתריע לך (ע"י צליל מעצבן והודעה קופצת) כאשר אם אתה או מישהו מהברית שלך(אם אתה הגנרל) תחת התקפה.
כאשר ההתרעה נדלקת אתה יכול לבחור מה לעשות באמצעות Ikariam+ תקיפה מתוסקרפטת!
איך להשתמש:
1) לך לScripts -> Check for attacks -> Check for attack on me/ally(שגרירות צריכה להיות פתוחה).
2) כאשר יש התקפה עליך בלבד Ikariam+ תקיפה מתוסקרפטת יותחל מידית על הקובץ "AttackedFight.ikr".

Ikariam+ תקיפה מתוסקרפטת

Ikariam+ תקיפה מתוסקרפטת זה כלי שתוקף (בוזז/פושט פיראטית/מעביר יחידות/וכד') על ידי הוראות מתוסרטות.
לדוגמה אם אתה רושם:
Set Hoplite 90
Set Ram 15
Raid
Wait 15
PiracyRaid
Set Hoplite 0
Set Ram 0
Set SteamGiant 200
Raid

זה יעשה:
* יגדיר 90 פלאנקסים ו-15 איילי ברזל
* יבזוז עם הצבא שהוגדר
* יחכה 15 דקותs
* יפשוט פיראטית על אותה עיר
* יגדיר ל-0 פלאנקסים, 0 איילי ברזל ו-200 ענקי קיטור
* יבזוז עם הצבא שהוגדר
כל כך קל, אתה יכול לתקוף אחרים מבלי בכלל להיות מחובר!
איך להשתמש:
1) לך לScripts -> Scripted attack -> New scripted attack (for new file) / Start script (open exist file)

איך לכתוב סקריפט

כל שורה חייבת להכיל רק פקודה אחת.
אל תיקח סקריפטים ממקורות שאינך סומך עליהם
כתיבת "#" או ";" בתחילת שורה תיחשב כהערה ותתעלם על ידי התוכנה.
לדוגמה />
; זוהי הערה
# זוהי גם כן הערה

רשימת פקודות מלאה (ללא מרכאות)

  1. "GoTo *Line*" - הסקריפט יחזור לשורה *Line* (החלף במספר שלם חיובי) ותמשיך משם
    לדוגמה:
    GoTo 1
    # The script will start from the beginning
  2. "Wait *Minutes*" - משהה את התוכנית ל*Minutes* דקות(החלף עם מספר שלם חיבוי)
    לדוגמה:
    Wait 10
    # The script will wait 10 minutes
  3. "RefreshData" - מרענן את הנתונים בחלון (אותו רענון קורה פעם בדקה)
    לדוגמה:
    RefreshData
    # The page will refresh
  4. "ChangeCityID *CityID*" - שנה עיר יחוס.
    לפי אופציה זו התוכנה תדע לאיזו עיר אתה מתכוון כאשר אתה רוצה לפשוט /לבזוז/וכד' על עיר
    כדי לקבל את המספר המזהה של העיר, לך למבט האי, וליד שם העיר יופיע המספר המזהה של העיר

    לדוגמה:
    ChangeCityID 86615
    # The selected city will set to city ID 86615
  5. "ChangeCurrentCityID *CityID*" - שנה עיר נוכחית.
    לפי אופציה זו התוכנה תדע מאיזו עיר אתה מתכוון כאשר אתה רוצה לפשוט /לבזוז/וכד' על עיר
    כדי לקבל את המספר המזהה של העיר, לך למבט האי, וליד שם העיר יופיע המספר המזהה של העיר
    העיר הזאת חייבת להיות בשליטתך, כלומר, אתה יכול לראות את העיר ברשימת הערים שלך.
    לדוגמה:
    ChangeCurrentCityID 86615
    # The current city will set to city ID 86615
  6. "UpgradeBuilding *PositionID*" - שדרג מבנה שנמצא באותו מיקום בעיר.
    המיקום של המבנה יהיה בתצוגת עיר ליד שם המבנה.
    לדוגמה:
    UpgradeBuilding 17
    # Upgrade pirate fortress
  7. "StartPiracy" - התחל יצור נקודות לכידה (צריך להיות בתצוגת עיר ובעיר עם מבצר פיראטים).
    לדוגמה:
    StartPiracy
    # Start piracy script
  8. "StopPiracy" - הפסק ייצור נקודות לכידה.
    לדוגמה:
    StopPiracy
    # Stop piracy script
  9. "Raid" - בזוז את העיר שנבחרה.
    לדוגמה:
    Raid
    # Raid the selected city
  10. "Occupy" - כבוש את העיר שנבחרה.
    לדוגמה:
    Occupy
    # Occupy the selected city
  11. "PiracyRaid" - פשוט פיראטית על העיר שנבחרה.
    לדוגמה:
    PiracyRaid
    # Piracy raid the selected city
  12. "DeployUnits" - פרוס יחידות בעיר שנבחרה.
    לדוגמה:
    DeployUnits
    # Deploy units into the selected city
  13. "ChangeView *View*" - שנה תצוגה.
    *View* יכול להיות: "City", "Island", "World"
    לדוגמה:
    ChangeView Island
    # Switch to island view
  14. "JS *Command*" - הרץ פקודת JavaScript (השתמש באפשרות זו רק אם אתה יודע מה אתה עושה!).
    לדוגמה:
    JS alert('this script work');
    # Run JavaScript pop-up message
  15. "Set *Unit* *HowMuch*" - לבחור איזו ויחידה וכמות.
    *HowMuch* - (החלף עם מספר שלם חיובי)
    *Unit* - Hoplite/SteamGiant/Spearman/Slinger/Swordsman/Archer/SulphurCarabineer/Ram/Catapult/Mortar/Gyrocopter/BalloonBombardier/Cook/Doctor/TransporterShips
    לדוגמה:
    Set Hoplite 100
    Set Mortar 30
    Set TransporterShips 50
    # Set to 100 hoplites, 30 mortars and 50 transporter ships
  16. "ResetUnits" - לאפס את בחירת כל היחידות.
    לדוגמה:
    ResetUnits
    # Equal to "Set *Unit* 0" for all units

Ikariam+ הרשמה אוטומטית

Ikariam+ הרשמה אוטומטית הוא כלי שמאפשר לך ליצור משתמש חדש (מאותה כתובת אימייל! זהו תקל במשחק) מהזמננה, ועושה לך את כל המדריך במשחק עד לרגע של בניית מבצר פיראטים.

דרישות מערכת

מערכת הפעלה: Windows 7/8/8.1/10 32&64 bit
.NET: 4.5.2
זיכרון זמן ריצה: 20MB (!)
זיכרון כונן קשיח: 100MB

הסכם שימוש

התוכנה פותחה למערכת ההפעלה ווינדוס. מאק ולינוקס לא נתמכות (למרות שאתה יכול להשתמש ב-Wine).
התוכנה לא אמורה לתת לך הרחקה מן המשחק. האפשרות היחידה לכך היא אם תגזים ותייצר מעל ל20,000 נקודות לכידה ביום.
במקרה של הרחקה מאיקרים, אני לא אחראי לכך.
